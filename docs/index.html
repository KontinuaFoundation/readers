About Kontinua Reader

Kontinua Reader is a powerful PDF reader application designed specifically for educational workbooks. Built for iOS devices and Android, it provides an intuitive interface with powerful annotation tools, digital resource integration, and navigation features that make studying more effective.

This alpha release is designed to gather user feedback and identify improvements as we prepare for our full public release. We appreciate your participation in testing Kontinua Reader iOS and providing your valuable insights.

Download Alpha Release

Kontinua Reader iOS is currently available as an alpha release for testing purposes. There are two ways to install the app:

iOS Downlaod

Option 1: TestFlight Installation (Recommended)

  1. Make sure you have TestFlight installed on your iOS device.
  2. Click on the TestFlight invitation link that was sent to your email.
  3. Follow the on-screen instructions to install Kontinua Reader iOS.
  4. Once installed, the app will appear on your home screen.

Request TestFlight Invitation

Option 2: Manual Installation for Developers

  1. Clone the repository from GitHub:
    2. git clone https://github.com/KontinuaFoundation/readers.git
  2. Open the project in Xcode:
    3. cd ReaderIOS open ReaderIOS.xcodeproj
  3. Build and run the project on your device or simulator.
